# 🚀 MultEdu - Jacob Trachtenberg System

> **Transformando o ensino da matemática através do Método Trachtenberg.**
> Focado em ajudar crianças a superarem a dificuldade com a tabuada por meio do cálculo mental rápido e lúdico. 🧮✨

---

### 🇧🇷 Sobre o Projeto / 🇺🇸 About the Project
* 🎓 **Objetivo / Goal:** Desenvolver uma plataforma interativa de alto desempenho para o ensino de aritmética.
* 🛠️ **Arquitetura / Architecture:** Sistema fullstack moderno utilizando Next.js 15 e Supabase BaaS.
* 🔐 **Segurança / Security:** Implementação rigorosa de políticas RLS e OpSec.
* 🏫 **Impacto Social / Impact:** Projeto voltado para alunos do Fundamental II da rede pública.
